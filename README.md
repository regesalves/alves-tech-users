# Sistema de Cadastro de Usuários

Sistema simples de cadastro e gerenciamento de usuários desenvolvido com Node.js, Express e SQLite.

---

## 📌 Sobre o projeto

Este projeto foi desenvolvido para praticar a integração entre interface web, backend e banco de dados, permitindo cadastrar, listar e excluir usuários de forma dinâmica.

---

## 🚀 Funcionalidades

- Cadastro de usuários
- Listagem de registros
- Exclusão de usuários
- Interface web integrada ao backend

---

## 💻 Tecnologias utilizadas

- Node.js
- Express
- SQLite
- HTML
- CSS
- JavaScript

---

## ▶️ Como executar

```bash
npm install
npm start
```

A aplicação estará disponível em:
http://localhost:3000/
